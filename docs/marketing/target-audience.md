# ターゲット市場

## 海外市場想定

### 主要ターゲット層

| ターゲット | 理由 |
|---|---|
| Undertale層 | 「敵視点」「メタ的構造」「感情的深度」への親和性 |
| Omori層 | 陰鬱・トラウマ・内面描写への親和性 |
| ダークJRPG層 | 重いテーマ・長編・戦略性を好む |
| villain perspective好き | 悪役主人公・動機の共感を求める層 |
| JRPG批評層 | 王道構造への批評的視点を楽しむ層 |

---

## 前作との差別化

| 要素 | 1 | 2 |
|---|---|---|
| 長さ | 短め | 長編 |
| 重さ | 中 | 重い |
| 陰鬱度 | 中 | 高 |
| 戦略性 | 低 | 重視 |

前作より深く・重く・長い作品として位置づける。

---

## 訴求ポイント

1. **villain perspectiveの徹底**：悪役が主人公で、その積み上げを追体験する
2. **王道への批評**：メタ的構造をプレイヤー自身が気づく設計
3. **感情的重さ**：静かな虚無感・解消されない違和感
4. **戦略的プレイ**：弟の戦術設計への没入

---

## 訴求時の注意

「王道批評ゲーム」として前面に出しすぎない。

違和感を露骨に強調しないという本作の設計思想をマーケティングにも適用する。
